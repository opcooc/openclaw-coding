# 绩效考核系统 - PRD 文档

**版本：** 1.0  
**创建时间：** 2026-04-16  
**状态：** 需求分析完成

---

## 1. 系统角色

| 角色 | 职责 | 权限 |
|------|------|------|
| **员工** | 被考核对象 | 填报目标任务、反馈实际完成、查看个人成绩 |
| **上级负责人** | 考核执行者 | 审核目标任务、对下级打分、查看部门汇总 |
| **管理员** | 系统配置 | 岗位/指标配置、考核周期管理、用户权限管理 |

---

## 2. 核心业务流程

### 2.1 月初填报流程（每月 1-5 日）

```
员工 → 填报当月 3-5 项重点任务 → 提交
          ↓
上级 → 审核确认（可修改） → 确认
          ↓
系统 → 锁定目标 → 进入执行期
```

### 2.2 月底打分流程（次月 1-5 日）

```
员工 → 反馈实际完成情况 → 提交
          ↓
上级 → 查看目标 vs 实际 → 评分
    ├─ 经营指标（系统自动计算/手工录入）
    ├─ 关键任务（手动评分）
    ├─ 团队管理（适用岗位）
    └─ 能力态度（手动评分）
          ↓
系统 → 计算总分 → 生成绩效等级
```

### 2.3 成绩查询流程

```
员工 → 查看个人历史成绩（月度/季度/年度）
上级 → 查看部门汇总报表（按人/按时间）
管理员 → 查看全公司统计
```

## 3. 数据结构

### 3.1 核心表结构

```sql
-- 岗位考核模板
perf_template (
  id,
  post_name,          -- 岗位名称
  weight_business,    -- 经营指标权重 (10-30)
  weight_task,        -- 关键任务权重 (50-65)
  weight_management,  -- 团队管理权重 (0-14)
  weight_attitude,    -- 能力态度权重 (20-30)
  created_at
)

-- 考核指标库
perf_indicator (
  id,
  template_id,        -- 关联模板
  dimension,          -- 维度：business/task/management/attitude
  indicator_name,     -- 指标名称
  default_weight,     -- 默认权重
  is_dynamic          -- 是否动态配置（重点任务）
)

-- 月度考核表
perf_assessment (
  id,
  employee_id,
  template_id,
  period,             -- 考核周期 (2026-04)
  status,             -- 状态：draft/submitted/confirmed/completed
  target_submit_date, -- 目标填报日期
  target_confirm_date,-- 目标确认日期
  result_submit_date, -- 结果反馈日期
  score_confirm_date, -- 打分确认日期
  total_score,        -- 总分
  performance_level,  -- 绩效等级
  created_at,
  updated_at
)

-- 目标任务记录
perf_target (
  id,
  assessment_id,
  indicator_id,
  target_value,       -- 目标值
  weight,             -- 权重
  description         -- 任务描述（重点任务用）
)

-- 实际完成记录
perf_result (
  id,
  assessment_id,
  indicator_id,
  actual_value,       -- 实际完成值
  score,              -- 得分
  comment             -- 评语
)

-- 评分记录
perf_score_record (
  id,
  assessment_id,
  scorer_id,          -- 评分人
  dimension,
  score,
  comment,
  scored_at
)
```

### 3.2 状态流转

```
draft → submitted → confirmed → completed
  │         │           │           │
  │         │           │           └─ 打分完成
  │         │           └─ 上级确认目标
  │         └─ 员工提交目标
  └─ 创建考核表
```
